# 二元分類與神經網路程式設計基礎筆記

## 1. 二元分類 (Binary Classification)
* **定義**：二元分類是一種演算法，旨在將輸入數據（如圖像）歸類為兩個類別之一。
* **範例**：貓咪辨識。輸入一張圖片，輸出標籤 $y$ 為 $1$（是貓）或 $0$（不是貓）。
* **演算法基礎**：本課程使用**邏輯回歸 (Logistic Regression)** 來傳達二元分類與神經網路的基本概念。

## 2. 圖像的表示與向量化 (Image Representation)
* **像素儲存**：電腦將圖像儲存為三個獨立的矩陣，分別對應 **紅 (Red)**、**綠 (Green)**、**藍 (Blue)** 三個顏色通道。
* **特徵向量化 (Unrolling)**：為了將圖像輸入模型，會將所有像素強度值「展開」成一個長特徵向量 $x$。
  * 若圖像大小為 $64 \times 64$ 像素，則特徵向量的維度 $n_x$（或簡稱 $n$）為：
    $$n_x = 64 \times 64 \times 3 = 12,288$$

## 3. 符號與資料結構 (Notation & Data Structures)
* **單一訓練範例**：表示為 $(x, y)$，其中 $x$ 是 $n_x$ 維的特徵向量，$y$ 為標籤（$0$ 或 $1$）。
* **樣本數量**：
  * $m$（或 $m_{\text{train}}$）：表示訓練集範例的總數。
  * $m_{\text{test}}$：表示測試集範例的總數。
* **矩陣表示法**：為了提高運算效率，會將多個範例整合進矩陣中：
  * **矩陣 $X$**：將所有訓練範例 $x^{(1)}, x^{(2)}, \dots, x^{(m)}$ 按**列 (column)** 堆疊而成。因此，$X$ 的形狀為 $(n_x, m)$。
  * **矩陣 $Y$**：將所有輸出標籤 $y^{(1)}, y^{(2)}, \dots, y^{(m)}$ 按**列**堆疊。因此，$Y$ 的形狀為 $(1, m)$。
* **慣例**：在神經網路實作中，將不同訓練範例的數據堆疊在不同「列」中，會使程式實作更為簡單與高效。

## 4. 神經網路程式設計重點
* **避免 Explicit For-loops**：在處理包含 $m$ 個範例的訓練集時，通常應避免使用明確的 `for` 迴圈，而是採用**向量化 (Vectorization)** 處理整個訓練集。
* **計算流程**：神經網路的計算通常組織為**正向傳播 (Forward Propagation)** 步驟，接著是**反向傳播 (Backward Propagation)** 步驟。
