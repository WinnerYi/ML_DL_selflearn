# 機器學習筆記：線性回歸與梯度下降

## 一、 模型參數與視覺化基礎

### 1. 線性回歸模型組成
在線性回歸模型 $f(x) = wx + b$ 中，參數的選擇直接決定了回歸線的形態與預測準確度：
* **截距 $b$**：代表回歸線與垂直軸（y 軸）的交點。
* **斜率 $w$**：決定了回歸線的傾斜方向與程度。
* **預測誤差**：透過測量**訓練資料點**與**回歸線**之間的**垂直距離**來計算。各點誤差的平方和即構成**代價函數（Cost Function）** $J(w,b)$。

### 2. 代價函數 $J(w,b)$ 的視覺化工具
為了理解參數如何影響模型擬合效果，可以使用以下兩種圖表：
* **等高線圖 (Contour Plots)**：由許多橢圓形組成。橢圓的**中心點**代表代價函數的**最小值（Minimum）**，也就是最佳擬合點。
* **3D 曲面圖 (3D Surface Plots)**：將代價 $J$ 呈現為一個立體曲面，可以旋轉觀察其起伏，幫助理解代價隨參數變化的趨勢。

### 3. 參數選擇與代價的對應關係
| 參數組合案例 | 回歸線狀態 | 代價函數 $J$ 的表現 | 在等高線圖上的位置 |
| :--- | :--- | :--- | :--- |
| **不佳的擬合** <br>*(例如 $w=-0.15, b=800$)* | 無法貼合資料點 | 非常高 | 遠離中心的點 |
| **水平線預測** <br>*(例如 $w=0, b=360$)* | 呈現水平狀 | 稍好，但仍非最佳解 | 介於中間偏遠位置 |
| **最佳擬合** | 穿過資料點分布中心 | 達到或接近最小值 | 非常接近最小橢圓的中心點 |

## 二、 梯度下降 (Gradient Descent) 核心概念

### 1. 什麼是梯度下降？
梯度下降是一種**系統化的最佳化演算法**，旨在透過自動調整參數（如 $w$ 和 $b$），找到使代價函數 $J$ 達到最小值的解。
* **應用範圍廣泛**：不僅用於線性回歸，也是訓練當今最先進**神經網路（深度學習）**的核心組件。
* **通用性高**：可以用於最小化任何函數，能處理擁有超過兩個參數的複雜模型。

### 2. 核心運作流程
> 💡 **形象化類比**：想像你被蒙上雙眼站在高爾夫球場的山丘上，目標是以最有效率的方式到達山谷底部。此時，**表面高度**代表成本函數值 $J(w,b)$，**經緯度軸**代表參數 $w$ 和 $b$。

1. **設定初始值**：從一組初始猜測值開始。在線性回歸中，通常直接將 $w$ 和 $b$ 都設為 `0`。
2. **尋找下降方向**：環顧四周 $360^{\circ}$，尋找能最快走下山谷的方向。
3. **採取小步幅 (Baby Steps)**：往**「最陡下降方向」（Steepest Descent）**邁出一小步。
4. **重複迭代**：到達新位置後再次尋找方向並邁步，直到抵達山谷底部（即局部最小值）。

### 3. 成本函數的形狀與局部最小值 (Local Minima)
* **線性回歸（平方誤差成本函數）**：表面通常呈現**「碗狀」**或**「吊床狀」**，結構相對簡單，通常只有一個全域最小值。
* **神經網路模型**：其成本函數表面非常複雜，包含多個起伏的高山與低谷。
* **局部最小值特性**：
    * **起始點**的不同可能會導致完全不同的結果（落入不同的山谷）。
    * 一旦演算法進入某個局部最小值，通常就會**停留在那裡**，而不會跳到另一個更低的山谷。

## 三、 數學實作與參數同時更新

### 1. 梯度下降演算法公式
梯度下降透過以下公式不斷更新模型參數 w 與 b：

$$w = w - \alpha \frac{\partial}{\partial w} J(w,b)$$

$$b = b - \alpha \frac{\partial}{\partial b} J(w,b)$$

* **賦值運算子 (`=`)**：在程式實作中代表將右側計算的結果儲存到左側的變數中（如 `a = a + 1`），而非數學上的等號。
* **學習率 $\alpha$ (Alpha)**：通常是一個介於 `0` 到 `1` 之間的小正數（例如 `0.01`），用來**控制步長**。$\alpha$ 越大步伐越激進；$\alpha$ 越小則像走小碎步。
* **導數項 (Derivative term)**：決定了向下走的方向，並與學習率共同決定每一步的大小。

### 2. 同時更新 (Simultaneous Update) 的重要性
在程式實作時，**必須同時更新參數 $w$ 與 $b$**，這是維持演算法正確性的關鍵細節。

* **正確的實作方式（使用暫存變數）**：
    ```python
   temp_w = w - alpha * d_dw * J(w,b)
   temp_b = b - alpha * d_db * J(w,b)
   w = temp_w
   b = temp_b
    ```
    *註：這樣可以確保在計算 $b$ 的更新值時，使用的是尚未被改變的 $w$。*
    

* **錯誤的實作方式**：
  ```python
  temp_w = w - alpha * d_dw * J(w,b)
  w = temp_w
  temp_b = b - alpha * d_db * J(w,b)    # 此處的 w 已經是更新後的值，會導致邏輯錯誤
  b = temp_b
  ```
    如果先計算並更新了 $w$，接著在計算 $b$ 的更新時直接帶入「已更新的 $w$」，這就不屬於標準的梯度下降法（雖然有時有效，但性質已改變）。

### 3. 收斂 (Convergence)
演算法會不斷重複上述更新步驟，直到達到**收斂**。所謂收斂是指參數 $w$ 和 $b$ 已經到達局部最小值，此時即使繼續疊代，參數的值也**不再有顯著變化**。

## 四、 導數項與學習率的直觀意義

### 1. 導數項（單一參數 $w$ 範例）
公式簡化為： $w = w - \alpha \frac{d}{dw} J(w)$。導數實質上就是該點上**切線的斜率（Slope）**，它能自動引導正確的方向：

* **情況 A：起點在最小值右側（斜率為正）**
    * 切線向右上傾斜，斜率 $\frac{d}{dw} J(w) > 0$。
    * 帶入公式：
      $$w = w - \alpha \({正數})$$
    * 結果：$w$ 會變小，在圖表上**向左移動**，朝向最小值靠近。
* **情況 B：起點在最小值左側（斜率為負）**
    * 切線向右下傾斜，斜率 $\frac{d}{dw} J(w) < 0$。
    * 帶入公式：
      $$w = w - \alpha \({負數})$$
    * 結果：$w$會變大，在圖表上**向右移動**，同樣朝向最小值靠近。

### 2. 學習率 $\alpha$ 的作用
* 學習率是一個**始終為正**的數值，決定了我們朝向最小值跨出的步伐大小。
* 當 $\alpha$ 選擇得當時，導數項負責引導正確方向，學習率確保適當步幅，兩者結合就能完美實現參數優化、降低成本。
