# 機器學習基礎與單變數線性回歸 核心筆記

## 一、 機器學習標準術語與符號

在機器學習中，掌握標準數學符號與專業術語是看懂公式與進行專業討論的基礎：

| 符號 / 術語 | 專業名稱 | 定義與說明 | 實例（以房價預測為例） |
| :--- | :--- | :--- | :--- |
| **訓練集**<br>*(Training Set)* | 訓練資料集 | 用於訓練模型、讓演算法學習的完整資料集。 | 包含所有房屋歷史成交紀錄的表格。 |
| $x$ | 輸入變數 / 特徵<br>*(Feature)* | 模型用來做預測的輸入數據。 | 房屋的坪數、大小。 |
| $y$ | 輸出變數 / 目標變數<br>*(Target Variable)* | 訓練集中的**實際真實值**，即模型試圖預測的結果。 | 房屋的實際售價。 |
| $\hat{y}$ | 預測值<br>*(y-hat)* | 模型對 $y$ 的預測或估計值（可能不等於真實值）。 | 模型估算出來的房價。 |
| $m$ | 樣本總數 | 整個訓練集中包含的資料總筆數。 | 資料集中共有 47 間房屋的資訊（$m = 47$）。 |
| $(x, y)$ | 單一訓練樣本 | 代表某一筆包含輸入與輸出答案的完整資料。 | 「30坪、售價1500萬」這某一筆資料。 |
| $(x^{(i)}, y^{(i)})$ | 第 $i$ 個訓練樣本 | 表示資料表中的第 $i$ 行（Index 索引）。<br>⚠️ **注意**：右上標 $(i)$ **並非數學上的冪次**。 | $(x^{(3)}, y^{(3)})$ 代表資料集中的第 3 筆樣本。 |

---

## 二、 監督式學習的運作流程

監督式學習演算法的核心邏輯是一個端到端的對應過程：

1. **資料輸入**：將包含「輸入特徵 $x$」與「真實目標 $y$」的**訓練集**餵給**學習演算法**。
2. **生成模型**：演算法透過學習數據，產生一個數學函數 $f$（即為**模型**，過去常被稱為 Hypothesis 假設）。
3. **進行預測**：當有新的輸入 $x$ 進來時，函數 $f$ 會計算並輸出一個預測值 $\hat{y}$。

$$\text{新輸入 } x \longrightarrow \text{模型 } f \longrightarrow \text{預測值 } \hat{y}$$

---

## 三、 單變數線性回歸 (Univariate Linear Regression)

### 1. 核心定義
* **線性回歸 (Linear Regression)**：使用直線函數對 $y$ 值進行預測的模型。
* **單變數 (Univariate)**：源自拉丁語（Uni = 單一，Variate = 變數），代表模型**只有單一個輸入特徵 $x$**（例如僅根據房屋大小來預測價格）。

### 2. 模型數學公式
在單變數線性回歸中，我們使用一條直線來表示函數 $f$：

$$f_{w,b}(x) = w \cdot x + b$$
*(亦可簡寫為 $f(x) = wx + b$)*

* **參數 $w$ 與 $b$**：決定這條直線的斜率與截距。演算法的目標是從數據中自動學習並優化這兩個參數，進而生成一條最適合數據分佈的直線（Best Fit Line）。
* **優化關鍵**：如何評估這條直線夠不夠準確？這取決於後續會討論到的**成本函數（Cost Function）**。

# 機器學習筆記：線性回歸與代價函數 (Cost Function)

## 1. 代價函數的定義與目的
* **核心目的**：代價函數用來衡量模型預測值與實際目標值（Target, $y$）之間的差距，藉此評估模型擬合數據的效果。
* **單一誤差 (Error)**：單一訓練樣本的誤差定義為預測值與實際值之差 $(\hat{y} - y)$.
* **平方誤差代價函數 (Squared Error Cost Function)**：這是線性回歸最常用的代價函數，公式通常寫作 $J(w,b)$。
* **計算方式**：將所有訓練樣本的誤差平方後加總，再取平均值。

---

## 2. 公式表達
$$J(w,b) = \frac{1}{2m} \sum_{i=1}^{m} \left( f_{w,b}(x^{(i)}) - y^{(i)} \right)^2$$

### 參數詳細說明：
| 符號 | 定義 | 說明 |
| :--- | :--- | :--- |
| $m$ | 訓練樣本總數 | 數據集中的資料筆數 |
| $x^{(i)}$ | 第 $i$ 個樣本的輸入特徵 | 模型的輸入變數 (Input Feature) |
| $y^{(i)}$ | 第 $i$ 個樣本的實際目標值 | 真實標籤 (Target / Label) |
| $f_{w,b}(x^{(i)})$ | 模型的預測值 $\hat{y}^{(i)}$ | 線性回歸中為 $f_{w,b}(x) = w \cdot x + b$ |
| $w$ | 權重 (Weight) | 決定直線斜率的參數 |
| $b$ | 偏差 (Bias) | 決定直線截距的參數 |

---

## 3. 慣例設計的背後原因
在公式中之所以使用 $\frac{1}{2m}$ 而非單純的平均，有其特殊的數學考量：
1. **除以 $m$**：確保代價函數的值**不會**僅因為訓練樣本數增加而變得無上限。它代表的是「平均平方誤差」，使不同規模的數據集具有可比性。
2. **除以 $2$**：這是一個純粹的**數學慣例**。當後續使用**梯度下降法 (Gradient Descent)** 對參數求偏導時，根據連鎖律，微積分的次方項 $2$ 會被拉到前面，剛好與 $\frac{1}{2}$ 互相抵消（約分），從而讓求導後的公式形式更加乾淨簡潔：
   $$\frac{\partial J}{\partial w} = \frac{1}{m} \sum_{i=1}^{m} \left( f_{w,b}(x^{(i)}) - y^{(i)} \right) \cdot x^{(i)}$$

---

## 4. 訓練目標 (Training Goal)
線性回歸的終極目標是找到一組最優的參數 $(w, b)$，使得代價函數 $J(w,b)$ 的值達到**最小**：

$$\min_{w,b} J(w,b)$$

* **直觀理解**：當 $J(w,b)$ 的值越接近 `0`，代表模型生成的直線與所有訓練數據點的總體距離（誤差）越小，即模型對現有數據的**擬合效果越好**。
